Bienvenus dans MARisk : Logiciel de management des risques des projets de construction

Pour ajouter un nouveau projet :

  1. Cliquez sur le bouton "Commencer un nouveau projet" ci-dessous.
  2. Remplissez les informations requises pour votre projet dans les champs prévus :
    • Nom du projet : Saisissez le nom de votre projet.
    • Description : Décrivez brièvement votre projet.
    • Date de début : Sélectionnez la date de début prévue pour votre projet.
    • Date de fin : Choisissez la date de fin attendue pour votre projet.
  3. Une fois que vous avez rempli tous les détails, cliquez sur le bouton "Enregistrer" pour ajouter votre projet.
